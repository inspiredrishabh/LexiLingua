# 🚀 Quick Start Guide - Legal Document AI Analyzer

**Get up and running in 5 minutes!**

## ⚡ Super Quick Setup

### 1. Install Everything

```bash
pip install -r requirements.txt
```

### 2. Get Your FREE Gemini API Key

1. Go to: https://makersuite.google.com/app/apikey
2. Sign in with Google account
3. Click "Create API Key"
4. Copy the key (keep it secret!)

### 3. Launch the App

```bash
streamlit run legal_app.py
```

### 4. Use the Tool

1. **Enter your API key** in the sidebar (secure, not stored)
2. **Upload your legal document** (PDF or image)
3. **Click "AI Analysis"** for complete breakdown
4. **Ask questions** about specific clauses

## 🎯 What You Get

### Instant Document Analysis

- **📋 Document Type**: Rental agreement, loan, etc.
- **🎯 Main Purpose**: What it's for in simple terms
- **💰 Financial Obligations**: All costs and fees
- **🚨 Red Flags**: Things to watch out for
- **📅 Important Dates**: Deadlines and renewals
- **🚪 Exit Options**: How to cancel or terminate

### Interactive Q&A

Ask questions like:

- "What happens if I pay late?"
- "How much will this cost me total?"
- "Can I get out of this contract?"
- "What are my rights here?"

### Multi-Language Support

- Upload documents in any language
- Get analysis in your preferred language
- Works with handwritten documents too!

## 🛠️ Troubleshooting

### Common Issues & Quick Fixes

**"No text extracted"**

- Try a clearer image/scan
- Make sure text is dark on light background
- Use PDF format if possible

**"API Key Error"**

- Double-check your Gemini API key
- Make sure you have internet connection
- Try creating a new API key

**"Installation Error"**

- Update pip: `python -m pip install --upgrade pip`
- Try installing packages one by one
- Use a virtual environment
